# DTI Project – Offspring Phenotype Prediction

This project predicts offspring phenotypes using parental phenotypes.

The system combines:
- Mendelian genetics rules
- Machine learning for polygenic traits
- Generative AI for synthetic dataset generation

ML predictions are used for:
- Height
- Skin tone

Other traits follow deterministic inheritance rules.

Technologies used:
- Python
- Scikit-learn
- Pandas
- NumPy
